# 🎬 Hispurpose Arts

A modern 3D animator portfolio showcasing cinematic motion graphics, VFX, product animations, and visual storytelling.

---

## 🚀 About the Project

Hispurpose Arts is a personal portfolio built to showcase high-quality animation work, including:

- VFX compositions  
- Product animations  
- Motion graphics  
- Cinematic short visuals  

The project is designed with smooth interactions, fast performance, and a clean studio-level aesthetic.

---

## 🛠️ Tech Stack

- Vite  
- React  
- TypeScript  
- Tailwind CSS  
- shadcn/ui  
- Framer Motion  
- React Router  

---

## 📦 Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate into the project
cd hispurpose-arts

# Install dependencies
npm install

# Start development server
npm run dev
